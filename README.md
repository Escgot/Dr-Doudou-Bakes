# Dr Doudou Bakes

A premium artisan dessert company website showcasing handcrafted pastries and traditional honey cakes with multi-language support (English, French, and Arabic).

## Features
- **Multi-language Support**: Seamless switching between English, French, and Arabic.
- **RTL Support**: Automatic Right-to-Left layout for Arabic.
- **Cinematic Experience**: High-end visual design with Framer Motion animations.
- **Mobile Responsive**: Fully optimized for all device sizes.
- **Recipe Gallery**: Exploration of artisanal recipes with detailed steps.

## Tech Stack
- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS, Lucide React icons
- **Animations**: Framer Motion
- **Context API**: Custom LanguageContext for global state management

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment
The project is configured for easy deployment on **Vercel** with custom routing handled in `vercel.json`.
